Functions

The 3 functions we can use are decleration, expression and arrow. We will primarly use the arrow function throughout this course.

  • Decleration

    • Created with the function keyword. They are hoisted, meaning they can be called before they are declared in the code.

  • Expression

    • Define functions as part of an expression. When you need to assign a function to a varibale or pass it as an argument to another function

  • Arrow

    • Provide a concise syntax for writing functions. Commonly used in array methods or as a concise callback function.
