JS Objects

I had some time after finishing tasks to get some research done on JavaScript Objects.

An example I found online compared JS Objects to a car. In real life, a car is an object. A car has properties like weight and color, and methods like start and stop. All cars have the same properties but the property values differ from car to car. All cars have the same methods, but the methods are performed at different times.

Objects are variables, but can contain many values.

It is common practice to declare objects with the const keyword.
